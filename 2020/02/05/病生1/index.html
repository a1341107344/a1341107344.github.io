水，钠代谢紊乱的分类

水，钠代谢紊乱常同时或相继发生，并相互影响，关系密切。

分类

  1. 脱水
    • 低渗性脱水
    • 高渗性脱水
    • 等渗性脱水
  2. 水中毒
  3. 水肿

    脱水（dehydration)

    指由于人体饮水不足或病变消耗大量水分，不能及时补充，导致细胞外液减少而引起新陈代谢障碍的临床症候群

    低渗性脱水/低容量性低钠血症

    特点：失Na多于失水，伴有细胞外液量减少

    原因和机制

  4. 经肾丢失
    1. 长期连续使用利尿药：呋塞米，噻嗪类药物抑制髓袢升支对Na的重吸收
    2. 肾上腺皮质功能不全：醛固酮分泌不足
    3. 肾实质性疾病：如慢性间质性肾疾患髓质间质破环
    4. 肾小管酸中毒（RTA）：肾小管排酸障碍，泌H+功能降低，H-Na交换减少
  5. 肾外丢失
    1. 经消化道失液：丧失大量消化液而只补充水分
    2. 液体在第三间隙积聚：胸水，腹水等
    3. 经皮肤丢失：大量出汗，只补充水分；大面积烧伤导致体液丢失

      对机体的影响

  • 细胞外液减少，易发生休克：低渗脱水状态下，水向胞内移动，血容量进一步减少，导致低血容量性休克
  • 血浆渗透压降低：无口渴感，难以自觉补充液体。血浆渗透压降低抑制渗透压感受器，ADH分泌减少，水的重吸收也减少
  • 有明显的失水体征：组织间液向细胞内转移，组织间液进一步减少，皮肤弹性减退，眼窝和婴儿囟门凹陷
  • 经肾失钠的低钠血症，尿钠含量增多，若是肾外因素所致，低血容量会激活肾素-血管紧张素-醛固酮系统，使尿钠含量减少

    防治

  • 防治原发病，去除病因
  • 适当补液
  • 原则上给予等渗液回复细胞外液血容量

    高渗性脱水/低容量性高钠血症

    特点：失水多于失钠

    原因和机制

  1. 水摄入减少：水源断绝，进食或饮水困难
  2. 水丢失过多
    • 经呼吸道丢失：过度性通气会使呼吸道黏膜不感性蒸发加强，持续过长时间会损失不含电解质的水分
      • 经皮肤失水：高热，大量出汗，甲亢时，会通过皮肤损失大量低渗液体。发热时，提体温每升高1.5℃。不感性蒸发量增加500ml
    • 经肾失水：中枢性尿崩症由于ADH分泌不足，肾性尿崩症时远曲小管和集合管对ADH反应缺乏时，肾排出大量低渗性液体
    • 经肠道丢失：呕吐，腹泻，消化道引流会导致等渗或低钠的消化液丢失

      对机体的影响

  • 口渴：细胞外液高渗，刺激渗透压感受器，引起口渴感
  • 细胞外液含量减少，尿量减少，尿比重增高：丢失细胞外液，ADH释放增多，水重吸收增加
  • 细胞内液向细胞外液转移：易导致细胞脱水使细胞皱缩
  • 血液浓缩
  • 中枢神经系统功能障碍：脑细胞脱水时，引起一系列中枢神经系统功能障碍，如嗜睡，肌肉抽搐，昏迷，甚至死亡。脑体积减小，颅骨与脑皮质血管张力增大，易静脉破裂

    防治

  • 防治原发病，去除病因
  • 补充缺少水分,不可口服者静脉滴入5%~10%葡萄糖
  • 适当补Na,给予生理盐水和葡萄糖混合液
  • 适当补K:胞内脱水使K排出,尿中排K增多

    等渗性脱水

    特点:钠和水成比例丢失,血容量减少

    可见于呕吐,腹泻,大面积烧伤,大量抽放胸水,腹水

    等渗性脱水若不进行处理,可通过不感性蒸发转化为高渗性脱水;若补给过多低渗液则导致低渗性脱水,临床上较少见单纯低渗性脱水

    水中毒(water intoxication)

    患者水潴留使体液量明显增多,血Na下降,但体钠总量正常或增多,又称高容量性低钠血症

    原因和机制

    主要是由于过多低渗性液体在体内潴留造成细胞内外液量都增多
  1. 水摄入过多:用无盐水灌肠,肠道吸收水分过多,精神性饮水过量和持续大量饮水.静脉输入含盐少或不含盐液体过快,超过肾脏排水能力
  2. 水排出减少:多见于急性肾衰竭,ADH分泌过多
    水中毒最常发生在急性肾功能不全的患者

    对机体的影响

  • 细胞外液量增加,血液稀释
  • 细胞内水肿：血钠浓度降低，水向细胞内转移，晚期会出现凹陷症状
  • 中枢神经系统症状：脑细胞肿胀，脑组织水肿使颅内压增高，脑脊液压力增加，出现一系列症状
  • 实验室检查可见血液稀释，血浆蛋白，血红蛋白浓度，血细胞比容降低，早期尿量增加
    防治
  • 防治原发病：急性肾衰竭，术后与心力衰竭患者，严格限制水摄入
  • 轻症患者，停止或限制水摄入
  • 重急性患者，严格限制进水，给予高渗盐水，静脉给予甘露醇等渗透性利尿剂

    水肿（edema）

    过多的液体在组织间隙或体腔内积聚

    分类

    按波及范围：全身性水肿，局部性水肿

    按发病原因：肾性水肿，肝性水肿等

    按发生水肿器官：皮下水肿，脑水肿等

    全身性水肿多见于充血性心力衰竭（心性水肿），肾病综合征，肾炎。局部性水肿多见于局部炎症

    水肿的发病机制

    血管内外液体交换平衡失调

    组织液生成和回流保持动态平衡，主要取决于有效流体静压，有效胶体渗透压，淋巴回流
  • 有效流体静压——驱使血管内液体向外滤出
  • 有效胶体渗透压——驱使液体回流入血管
  • 淋巴回流——组织液回流剩余部分须经淋巴回流进入血液循环
  1. 毛细血管流体静压升高

    导致有效流体静压升高，有效滤过压增大，超过淋巴回流代偿能力，组织液生成增多
    升高原因：静脉压升高，充血性心力衰竭时静脉压增高导致全身性水肿。肿瘤压迫静脉时毛细血管静脉压增高
  2. 血浆胶体渗透压降低

    血浆蛋白减少，主要由于：蛋白质合成障碍（肝硬化）、蛋白质丧失过多（肾病综合征），蛋白质分解代谢加强（恶性肿瘤）
  3. 微血管壁通透性增加

    蛋白质进入组织，血浆胶体渗透压下降，水肿液中蛋白质含量高
  4. 淋巴回流受阻

    体内外液体交换平衡失调——钠，水潴留

  5. 肾小球滤过率下降：
    肾小球滤过减少，不伴有重吸收减少时，就会导致钠，水潴留

    下降原因：
  • 广泛的肾小球病变
  • 有效循环血量明显减少
  1. 近曲小管重吸收钠，水增多：

    有效循环血量减少时，近曲小管重吸收钠，水增多，是全身性水肿发病的重要原因
  • 心房钠尿肽分泌减少：心房牵张感受器兴奋性降低，ANP分泌减少，钠水重吸收增加
  • 肾小球滤过分数增加：肾内因素作用，肾小球滤过分数=肾小球滤过率/肾血浆流量
